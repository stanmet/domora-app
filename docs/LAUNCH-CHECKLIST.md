# Пусковой чек-лист Domora: от кода до первых денег

## 1. Бизнес и юридика (делаете лично вы, параллельно с разработкой)

Регистрация бизнеса в Ирландии: sole trader через Revenue (быстро, для старта) или Limited Company через CRO (около 50 евро, солиднее для Stripe и партнеров). Открыть бизнес-счет в банке (Revolut Business / AIB / BOI). Зарегистрировать домен (.ie требует подтверждения связи с Ирландией, .com проще).

Stripe: создать аккаунт на stripe.com, пройти активацию бизнеса, включить Connect (тип Express) в настройках. До активации все работает в тест-режиме с тестовыми картами, для разработки этого достаточно.

Документы (юрист, 1-2 недели): Terms of Service с агентской моделью (платформа не сторона сделки, исполнители независимые подрядчики), Privacy Policy под GDPR, Cancellation Policy с тирами из спецификации, Cookie Policy. Назначить процесс обработки запросов GDPR (экспорт и удаление данных).

Страхование платформы: professional indemnity + cyber. Гарантийный фонд споров: завести отдельный резерв (рекомендация: 2% GMV).

## 2. Инфраструктура (30 минут, бесплатные тиры)

База: создать проект в Supabase (Postgres + Auth + Realtime + Storage в одном). Хостинг: Vercel, подключить репозиторий. Скопировать .env.example в переменные окружения Vercel. Вебхуки Stripe: два классических (snapshot) destination на https://ваш-домен/api/stripe/webhook, thin-формат не подходит; пошаговая инструкция в docs/stripe-webhook-setup.md. DeepL API: бесплатный тир 500 тыс. символов в месяц.

## 3. Разработка по спринтам (кодовая база из этого пакета - фундамент)

Спринт 1, недели 1-2: аутентификация (Supabase Auth: email + Google), профили, каталог и плашки с модерацией. UI уже готов из прототипов, его переносим в Next.js.
Спринт 2, недели 3-4: флоу брони на живом Stripe в тест-режиме (route-файлы из этого пакета), кабинет исполнителя, принять/отклонить.
Спринт 3, недели 5-6: чат с автопереводом и фильтром контактов, статусы выполнения, фото до/после.
Спринт 4, недели 7-8: отмены по тирам, возвраты, отзывы слепой публикацией, воркеры-таймеры (jobs.ts).
Спринт 5, недели 9-10: админка (модерация, споры, страйки), выплаты на живом Stripe, e2e-тесты флоу денег.
Недели 11-12: закрытая бета с 10-15 исполнителями в Дублине, реальные заказы друзей, шлифовка.

## 4. Тестирование денег (критично, до запуска)

Тестовые карты Stripe: 4242 4242 4242 4242 (успех), 4000 0000 0000 9995 (отклонение). Прогнать сценарии: холд -> accept -> capture -> completed -> transfer; холд -> decline -> отмена холда; холд -> 72 часа -> авто-EXPIRED; capture -> спор -> заморозка transfer -> частичный возврат; chargeback -> заморозка. Каждый сценарий должен сходиться до цента в таблицах payments/transfers/refunds.

## 5. Критерий "готово к работе"

Живой Stripe активирован, юрдоки опубликованы, 5+ исполнителей в каждой стартовой категории прошли верификацию, e2e-сценарии денег зеленые, админ может за 5 минут: одобрить плашку, вернуть деньги, заморозить профиль. После этого включаем рекламу.
